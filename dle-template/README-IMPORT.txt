==================================================================
  ШАБЛОН "НОВОСТИ" (дизайн Digg) для DataLife Engine v20.0
  Инструкция по установке
==================================================================

1. СТРУКТУРА ФАЙЛОВ
--------------------------------------------------
Загрузите папку шаблона на хостинг в:
    /templates/Novosti/

Содержимое:
    main.tpl          — каркас сайта: шапка, лента, сайдбар, футер
    shortstory.tpl    — карточка новости в ленте/категории
    fullstory.tpl     — страница полной новости + комментарии
    comments.tpl      — оформление одного комментария
    login.tpl         — форма входа
    registration.tpl  — форма регистрации
    userinfo.tpl      — профиль пользователя (аватар, публикации, закладки)
    profile.tpl       — настройки аккаунта (изменение профиля)
    css/style.css     — все стили (дизайн Digg)

2. АКТИВАЦИЯ ШАБЛОНА
--------------------------------------------------
Админка → Настройки → Основные → "Шаблон сайта" → выбрать "Novosti" → Сохранить.

3. ИСПОЛЬЗОВАННЫЕ ВОЗМОЖНОСТИ ДВИЖКА
--------------------------------------------------
[Категории и навигация]
    {categorymenu}   — меню категорий в шапке
    [category]...     — вывод категории новости
    {navigation}      — постраничная навигация (пагинация)

[Рейтинг и просмотры]
    [rating]{rating}[/rating]   — рейтинг новости (кнопка "↑" как в Digg)
    {news-views}                — счётчик просмотров
    [com-rating]{com-rating}    — рейтинг комментария

[Комментарии DLE]
    {addcomments}    — форма добавления комментария
    {comments}       — список комментариев (рендерится через comments.tpl)
    {reply-id}       — кнопка "Ответить"

[Поиск, теги, пагинация]
    Форма поиска     — index.php?do=search (в шапке)
    {tagscloud}      — облако тегов в сайдбаре
    [tags]{tags}     — теги внутри новости
    {topnews}        — популярные новости (блок "Тренды")
    {related-news}   — похожие материалы под статьёй

4. СООТВЕТСТВИЕ ДИЗАЙНА И ТЕГОВ DLE
--------------------------------------------------
  Блок дизайна (React)        →  Тег / файл DLE
  ----------------------------------------------------
  Featured-баннер             →  fullstory.tpl (главная новость)
  Карточка новости            →  shortstory.tpl
  Заголовок карточки          →  {title} + {full-link}
  Категория (цветной тег)     →  [category]{category}[/category]
  Краткий текст               →  {short-story}
  Полный текст                →  {full-story}
  Лид/описание                →  {description}
  Автор                       →  {author}
  Дата                        →  {date}
  Кнопка Digg (↑ + число)     →  [rating]{rating}[/rating]
  Просмотры (👁)              →  {news-views}
  Превью-картинка             →  [thumb]{thumb-storyimage}[/thumb]
  Блок "Тренды"               →  {topnews limit="5"}
  Облако тегов                →  {tagscloud}
  Похожие материалы           →  {related-news}
  Комментарии                 →  {comments} + comments.tpl
  Форма комментария           →  {addcomments}
  Лайк комментария (♥)        →  [com-rating]{com-rating}[/com-rating]
  Пагинация                   →  {navigation}
  Поиск                       →  index.php?do=search

5. ЦВЕТА (можно менять в css/style.css → :root)
--------------------------------------------------
  --accent: #ff5e2c   — оранжевый акцент (кнопки, рейтинг)
  --foreground: #141414 — основной чёрный
  --background: #fafafa — фон страницы

6. ТЁМНАЯ ТЕМА
--------------------------------------------------
  Кнопка "🌙 / ☀️" в шапке переключает светлую/тёмную тему.
  Выбор сохраняется в localStorage браузера, при первом входе
  учитывается системная тема пользователя.
  Цвета тёмной темы — в css/style.css → html.dark { ... }
  Логика — JS в конце main.tpl (без зависимостей).

7. ПОЛЬЗОВАТЕЛИ (вход / регистрация / профиль)
--------------------------------------------------
Используется ШТАТНАЯ система пользователей DLE — отдельная база
данных не нужна, всё работает через движок.

  Файл               Страница DLE
  ----------------------------------------------------
  login.tpl          index.php?do=register (вкладка входа)
  registration.tpl   index.php?do=register&doaction=newuser
  userinfo.tpl       user/{login}/ — профиль пользователя
  profile.tpl        index.php?do=cp — настройки аккаунта

  Использованные теги:
  {login}            — логин/имя пользователя
  {foto} [avatar]    — аватар (загруженное фото)
  {info}             — поле "О себе"
  {registration}     — дата регистрации
  {news-num}         — кол-во публикаций
  {comments-num}     — кол-во комментариев
  {user-rating}      — рейтинг пользователя
  {user-news-list}   — список публикаций в профиле
  {sec_code}         — капча на регистрации
  {socialauth}       — соц. авторизация (если включена)
  do=favorites       — закладки пользователя
  action=logout      — выход из аккаунта
